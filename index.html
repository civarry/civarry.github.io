macos · ai

Lasso

Circle to Search, but native to the Mac. Hit a hotkey, drag a box around anything on screen — copy the text out of it, translate it in place, ask AI about it, identify objects, scan QR codes, pick colors. Local by default: Apple Intelligence and Ollama run on-device, and nothing leaves the machine unless you opt in. Shipped as a signed menu-bar app with 15+ releases.

Swift Apple Intelligence Ollama Vision OCR
real-time

MotionCast

Turns a phone into a real-time motion source for any computer — orientation, gyro, and two-way haptics stream over a local HTTPS + WebSocket link. Quaternion-based and gimbal-lock free, with per-device auto-calibration. Installs as a PWA; zero frontend frameworks.

JavaScript WebSocket PWA

Android File Transfer

A macOS desktop app for managing Android files over ADB. Built because the official tool is terrible. Drag-and-drop, batch operations, clean UI.

Python PyQt6 ADB
